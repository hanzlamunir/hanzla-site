
# Intro

# What I Do

At Informly, I build reusable React component libraries that have reduced UI duplication by ~40% across our product. I focus on UI performance — through component refactoring, memoization, and code-splitting I've helped cut initial render times by ~30%. I implement centralized state management with Redux and Context API across 15+ screens, and I set up modern frontend tooling with Vite to give the team 3x faster local builds.

# How I Work

I follow a component-driven development approach: small, composable pieces that are easy to reason about, test, and reuse. AI is part of my daily workflow — I lean on Cursor, GitHub Copilot, and Claude to move faster on scaffolding, refactoring, and debugging, while keeping ownership of design decisions. I work closely with backend engineers to align on REST API contracts and deployment processes so the UI and the services behind it ship together cleanly.
